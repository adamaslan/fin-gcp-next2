## Typical Setup Flow

1. **Prerequisites Check**
   - Do you have Node.js/npm installed?
   - Do you have Python 3.11+?
   - Do you have git?
   - See: [`first-run-checklist.md`](first-run-checklist.md)

2. **Choose Your Path**
   - If you have <5 minutes: [`quickstart-5min.md`](quickstart-5min.md)
   - If you want full details: [`development-setup.md`](development-setup.md)

3. **Environment Setup**
   - Clone the repository
   - Install dependencies
   - Configure environment variables
   - See: [`mamba-environment.md`](mamba-environment.md)

4. **Verify Installation**
   - Run the verification checks
   - All systems green?
   - See: [`first-run-checklist.md`](first-run-checklist.md)

5. **Choose Your Focus**
   - Frontend dev? Go to: [`../frontend/`](../frontend/)
   - Backend dev? Go to: [`../backend/`](../backend/)
   - DevOps? Go to: [`../devops/`](../devops/)
   - Architecture? Go to: [`../architecture/`](../architecture/)
